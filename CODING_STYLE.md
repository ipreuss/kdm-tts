# Coding Style Guide

For comprehensive Lua coding conventions, see the **`kdm-coding-conventions`** skill (auto-loads when writing or reviewing code).

The skill covers:
- Clarity first, fail-fast philosophy
- Function contracts and error handling
- SOLID principles and refactoring expectations
- Module export patterns
- Naming conventions and paradigm preferences
- Return values for async operations

For workflow expectations, see `PROCESS.md`.
