# Deck Layout Positions

This document describes the physical layout of card decks on the game boards.

## Settlement Board - Top Row

The top row of decks on the Settlement Board follows a consistent grid pattern.
Each deck has a reset button positioned to its left.

From left to right:

| Position | Deck | LocationData Key |
|----------|------|------------------|
| 1 | Abilities | "Ability" |
| 2 | Fighting Arts | "Fighting Arts" |
| 3 | Secret Fighting Arts | "Secret Fighting Arts" |
| 4 | Disorders | "Disorder" |
| 5 | Severe Injuries | "Severe Injury" |
| 6 | Tactics | "Tactics" |
| 7 | Seed Patterns | "Seed Patterns" |
| 8 | Patterns | "Patterns" |

## Settlement Board - Second Row

| Position | Deck | LocationData Key |
|----------|------|------------------|
| 1 | Pattern Gear | "Pattern Gear" |
| 2-8 | (unused) | - |

## Coordinate System

- X axis: Left (-) to Right (+)
- Z axis: Back (-) to Front (+)  
- Y axis: Down (-) to Up (+)

Each deck slot is approximately 2.67 units apart on the X axis.

## Button Offset

Reset buttons are positioned to the left of their corresponding deck.
The deck spawns with its center offset from the button's top-left corner.
